## 
Problem Statement

You are given the Wisconsin Diagnostic Breast Cancer (WDBC) Data Set; the dataset can be found in wdbc.data and the description is found in wdbc.names.

The data set consists of 32 attributes and 569 instances. The first attribute is the instance Id. The second attribute represents the class label, Diagnosis (M = malignant, B = benign). The rest of the 30 attributes are real-valued features that are computed for each cell nucleus. Note that the mean, standard error (SE) and "worst" or largest (mean of the three largest values) of the main features were computed for each image, resulting in these 30 features. For instance, field 3 is Mean Radius, field 13 is Radius SE, field 23 is Worst Radius.

The features are computed from a digitized image of a fine needle aspirate (FNA) of a breast mass. They describe the characteristics of the cell nuclei present in the image. You are required to explore this dataset through data exploration and visualization techniques.
